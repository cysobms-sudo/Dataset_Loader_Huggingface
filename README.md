📊 Hugging Face Dataset Loader

یک برنامه حرفه‌ای و کاربرپسند برای بارگذاری، مشاهده و مدیریت دیتاست‌های Hugging Face با رابط کاربری گرافیکی (GUI)

 ✨ ویژگی‌ها
🚀 بارگذاری آسان:  بارگذاری دیتاست‌ها از Hugging Face با چند کلیک ساده 
📊 نمایش اطلاعات کامل:  مشاهده جزئیات دیتاست شامل تعداد ردیف، ستون‌ها، نوع داده‌ها و توضیحات 
📋 پیش‌نمایش داده:  نمایش چند ردیف اول داده‌ها به صورت جدولی و مرتب 
💾 ذخیره‌سازی حرفه‌ای:  ذخیره اطلاعات در فایل‌های TXT،  CSV و Excel 
🎨 رابط کاربری دوستانه:  طراحی زیبا با قابلیت تغییر تم (شب/روز) 
⚙️ تنظیمات پیشرفته:  شخصی‌سازی کامل برنامه بر اساس نیاز کاربر 
📜 تاریخچه هوشمند : ذخیره دیتاست‌های اخیر برای دسترسی سریع‌تر 
🔄 لغو: عملیات  امکان لغو بارگذاری در هر مرحله

نکته مهم: پس اجرای برنامه ممکن است حدود دو دقیقه برای بارگذاری کامل زمان لازم داشته باشد
